**teKha_eatsCookies** is a lightweight, efficient browser extension designed for Chromium-based browsers (Chrome, Edge, Brave, etc.). It allows developers and power users to seamlessly export and import cookies for major platforms including Google, Facebook, GitHub, and more.

Whether you are migrating sessions between browsers or managing multiple developer environments, **teKha_eatsCookies** provides a clean interface to handle cookie data securely and quickly.

---

## Features

* **Multi-Platform Support**: Works flawlessly with Google, Facebook, GitHub, and any other site.
* **Chromium Optimized**: Built specifically for the Chromium engine for maximum compatibility.
* **One-Click Export**: Quickly grab your current session cookies in a portable format.
* **Seamless Import**: Restore sessions instantly by importing cookie data.
* **Lightweight & Fast**: Minimal resource footprint with a clean, intuitive popup UI.

---

## Tech Stack

* **Frontend**: HTML, CSS
* **Logic**: JavaScript (Chrome Extension API)
* **Manifest**: Version 3 (MV3) compliant

## Installation Guide (Local Setup)

To install the extension using the provided .rar release file, follow these steps:

### 1. Extract the Files
* Download the `teKhaEatsCookies v0.1.0_winx64.rar` file from the [Releases](https://github.com/pamindu-fernando/teKha_eatsCookies/releases) section.
* Extract the contents of the .rar file into a dedicated folder on your computer (e.g., `C:\Extensions\teKhaEatsCookies v0.1.0_winx64`).

### 2. Open Browser Extensions Page
* Open your Chromium-based browser (Google Chrome, Microsoft Edge, Brave, etc.).
* In the address bar, type the following and press Enter:
    * **Chrome**: `chrome://extensions`
    * **Edge**: `edge://extensions`
    * **Brave**: `brave://extensions`

### 3. Enable Developer Mode
* In the top-right corner of the Extensions page, toggle the **Developer mode** switch to **ON**.

### 4. Load the Extension
* Click the **Load unpacked** button that appears.
* Navigate to the folder where you extracted the .rar file.
* Select the folder (the one containing the `manifest.json` file) and click **Select Folder**.

### 5. Verify Installation
* The **teKha_eatsCookies** icon should now appear in your extensions list.
* Pin it to your toolbar for easy access by clicking the extension icon next to your profile picture.

## Usage

1. Click on the **teKha_eatsCookies** icon in your browser toolbar.
2. **To Export**: Navigate to the site you want cookies from, open the extension, and use the Export function.
3. **To Import**: Open the extension on the target site and paste your cookie data into the provided field.

---

## Security Warning

**Important**: Cookies contain sensitive session information. Never share your exported cookie strings with anyone, as they can be used to gain unauthorized access to your accounts.
